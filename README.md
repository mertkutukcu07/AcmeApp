 What is Acme?

 Acme is a mobile application with Turkish and English language options where you can register and view job adverts and apply.

How to Operate?
1 - install dependencies

Usage yarn: yarn

2 - Start the expo

npx expo start -c

a key press for Android 
i key press for iOS
